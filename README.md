## 🔧 Personalizações

Você pode facilmente personalizar:

### Cores
Edite o `tailwind.config` no `<head>`:
```javascript
colors: {
  brand: "#E11D24",      // Altere a cor primária
  brandDark: "#B91C1C",
  surface: "#111111",
  card: "#1A1A1A",
}
```

### Fontes
Edite o `fontFamily`:
```javascript
fontFamily: {
  heading: ["Oswald", "sans-serif"],
  body: ["Source Sans 3", "sans-serif"],
}
```

### Imagem do Produto
Substitua o `src` da imagem:
```html
<img src="sua-imagem-aqui.webp" alt="Kit Whey" />
```

### Links de Afiliação
Atualize os `href` dos botões:
```html
<a href="seu-link-shopee">COMPRAR NA SHOPEE</a>
<a href="seu-link-mercado-livre">COMPRAR NO MERCADO LIVRE</a>
```
